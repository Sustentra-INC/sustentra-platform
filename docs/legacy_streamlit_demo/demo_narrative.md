# Demo Narrative

This five-minute walkthrough shows the prepared auditor workflow and handoffs between evidence, calculations, gaps, and regulatory support.

1. Configure audit scope.
- Open Audit Setup.
- Review company profile, reporting boundary, regulation settings, and methodology defaults.
- Save the setup to session state.

2. Upload workbook and evidence.
- Open Evidence Intake.
- Upload one workbook and supporting evidence files.
- Select the prepared scenario and run the prepared workflow.

3. Review extracted fields against original source.
- Open Extraction Review.
- Select evidence by status filters.
- Compare source snippet/preview on the left and extracted fields on the right.
- Apply Accept/Edit/Reject/Mark unclear actions.

4. Watch validation checks.
- Open Validation.
- Select a validation record.
- Review deterministic pass/fail checks and next-check routing.

5. Review recalculation and reconciliation.
- Open Calculation & Reconciliation.
- Trace activity -> factor -> gas-level output -> CO2e -> workbook variance.
- Confirm not-computed records are explicitly labeled with reasons.

6. Open three representative gaps.
- Open Gap Analysis.
- Review one numerical mismatch finding.
- Review one missing-support finding.
- Review one methodology/GWP-basis finding.
- Trigger Open evidence / Open workbook / Ask Regulatory Assistant actions.

7. Ask the Regulatory Assistant.
- Open Regulatory Assistant.
- Ask a contextual gap question.
- Review Issue, Evidence trace, Audit context, Regulatory basis, and Next actions.
- Confirm fallback notice appears when live service is unavailable.
