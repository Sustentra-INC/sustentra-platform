# Pilot Scope

## In Scope (Pilot 1)

- Create engagements.
- Upload and register documents.
- Trigger extraction processing runs.
- Show extracted candidates with source references.
- Capture reviewer decisions and approved values.
- List approved evidence by engagement.

## Out of Scope (Pilot 1)

- Full validation engine parity.
- Full calculation and reconciliation parity.
- Full gap ticket automation parity.
- Production orchestration and queue tuning.

## Success Criteria

- Evidence can be uploaded and processed end-to-end.
- Every reviewed field has traceability to source snippets.
- Approved evidence can be retrieved through backend APIs.
