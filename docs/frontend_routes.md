# Frontend Route Plan

- `/` : product overview and pilot quick links.
- `/audit-setup` : engagement setup and scope metadata.
- `/evidence-intake` : upload status and document registry.
- `/extraction-review` : candidate review workspace with source snippets.
- `/validation` : placeholder for downstream validation results.
- `/calculation` : placeholder for calculation/reconciliation phase.
- `/gap-analysis` : placeholder for gap findings and clarifications.
- `/assistant` : assistant interaction shell.

## Route Strategy

Pages are intentionally lightweight and composed from reusable components under `frontend/components/`.
